#### 安装步骤

1. `git clone https://gitee.com/MarcoMaHH/rent8_wechat.git`

2. `npm install`

3. 安装完之后，需要在微信开发者工具中对 npm 进行构建：工具 - 构建 npm

4. 根据后端访问地址，修改app.js文件

5. 修改appid
